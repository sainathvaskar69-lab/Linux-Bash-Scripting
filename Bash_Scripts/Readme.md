# Bash Scripting for Cybersecurity Practice

##  Overview
This repository contains **Bash scripting exercises focused on cybersecurity use-cases**.

The scripts are written to build **practical Linux automation skills** required in real security roles, not academic scripting.

Bash is widely used in:
- SOC environments
- Incident response
- Threat hunting
- System hardening
- Red team tooling
- Blue team automation

This repository exists to train that mindset.

---

##  Security-Relevant Concepts Practiced
- Linux file and directory analysis
- Log and file monitoring
- Disk and system resource checks
- Automation of repetitive security tasks
- Conditional logic for alerting and validation
- Safe handling of system paths and permissions
